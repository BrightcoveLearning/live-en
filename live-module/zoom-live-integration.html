Integrating a Zoom meeting into Brightcove Live

The Brightcove Live Zoom integration allows to you to broadcast Zoom meetings easily using Brightcove Live.

Prerequisites

In the Zoom Marketplace

Zoom integration with Brightcove Live is managed by the Brightcove Live Connect app in the Zoom Marketplace. It is an account-level app intended for use by licensed Owner or Admin-level Zoom accounts. Please ensure your Zoom account has the correct privileges to support the integration.

In Brightcove Video Cloud

Before starting an integration, you must have a custom metadata field to track whether an event is streaming from Zoom. This will track the state of the integration between sessions, i.e., if an event is currently linked to a Zoom meeting. For specific details on how to add the custom metadata field, please review the following documentation.

  1. Login to Video Cloud Studio.
  2. In the left navigation, click Admin > Video Fields.
  3. Click Add Custom Field.
  4. Enter a Display Name for your custom metadata field.
  5. For the Internal Name field, you should use: bc_meeting_id
    Don't add any external Zoom Id or data into the custom field.
  6. Enter a Description for the custom metadata field.

Best Practices

For best results, please follow the directions below.

  1. Use the Start and Stop Broadcasting buttons in the control room rather than the host Zoom player.
  2. Avoid streaming multiple Zoom meetings at the same time under a single account.

Creating a Zoom Meeting

  1. First, access your Zoom account, and create a Zoom meeting. For information on how to create a Zoom meeting, please follow the Zoom Documentation.
  2. In Zoom, go to Settings > Meeting (Advanced) > live streaming meetings.
  3. Make sure your Zoom account settings have the Custom Live Streaming Service enabled.

Integrating the Zoom Account with Brightcove Live

To integrate a Zoom account with Brightcove Live, follow these steps:

  1. Login to Video Cloud Studio.
  2. Open the Admin module.
  3. In the left navigation, click Integrations.
  4. Select the Integrations option from the dropdown menu.
  5. Click the Add Integration button to link one or more Zoom accounts to Brightcove Video Cloud.
  6. A pop-up window requesting access to your Zoom account will appear. Click the Authorize button.
  7. Provide your Zoom credentials to integrate the account.

Adding the Zoom Integration to a Brightcove Live event

To add the Zoom integration to Brightcove Live event, follow these steps:

  1. Navigate to the Live module in Video Cloud.
  2. Click the Create Event button on the Live Manage Events page. For complete information on how to create a Live event please review the Creating and Managing Live Events using the Live Module documentation.
  3. Fill in the Live event details as normal, until you reach the Integrations section.
  4. Click the Add Integration button.
  5. You will be able to see a list of your active and scheduled Zoom meetings.
  6. Click the dropdown menu.
  7. Select your meeting. In this example, I am choosing the Test Docs Meeting.
    Only one integration can be added to an event at a time. To clear the integration click on the Remove Integration button. If you want to have more than one account, you will be able to do so in the Admin > Integrations panel.
  8. Once ready to create the Live event, click on the Create Event button.
  9. In the Control Room, there will be a tabbed option to stream your meeting from Zoom. If you prefer not to use the integration source, switch back to the Ingest URL tab in the top left part.
  10. If your Zoom meeting was recently created, try syncing with Zoom again using the Sync button.
    If your meeting is still not showing up after syncing, try verifying that your meeting is Scheduled in Zoom and that you have fewer than 30 meetings scheduled.
  11. Head on to your Zoom meeting account.
  12. Click the blue Start button to start your meeting.
    Due to a Zoom limitation, before starting to broadcast the Zoom meeting to Brightcove Live, the Zoom meeting has to be started.
  13. Back In the Brightcove Live Control Room, click the Start Broadcasting button.
  14. After the last step, you will be able to see your Zoom Meeting in the Control Room.
    • At any time during the Zoom Meeting, you can stop and restart broadcasting. However, if you are not actively broadcasting the event may time out if the gap exceeds the Event reconnect time.
    • Zoom broadcast over Brightcove will be delayed by approximately 20 seconds.
    • If the Zoom meeting does not start streaming after 20 seconds you can click Stop Broadcasting and then Start Broadcasting again.
    • Because the Zoom stream is a third-party service, some advanced Live features may not be supported, including SSAI.

Uninstalling the Zoom Integration

Remove the integration from Video Cloud

To remove Zoom integrations from your account within Brightcove:

  1. Log in to Video Cloud Studio.
  2. Click the Admin button in the navigation header.
  3. Select the Integrations option from the dropdown menu.
  4. Click the Trash icon.
  5. When asked to confirm, click the Confirm button.

To remove Brightcove Live Connect from your Zoom account

You can completely remove the Brightcove Live Connect app from your Zoom account by connecting to Zoom:

  1. Login to your Zoom Account and navigate to the Zoom App Marketplace.
  2. Click Manage > Installed Apps or search for the Brightcove app.
  3. Click the Brightcove app.
  4. Click Uninstall.
